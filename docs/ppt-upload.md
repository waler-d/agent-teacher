# 如何上传 PPT 到知识库

Agent Teacher 通过**飞书**接收 PPT，自动提取文字并拆成知识点入库。

---

## 前提

1. 机器人已添加到你所在的飞书群或私聊
2. 飞书应用已开通权限：**im:message**、**im:message:send_as_bot**、**im:resource**（下载文件）
3. Vercel 已部署最新版 `agent-teacher`

---

## 方式一：直接上传文件（推荐）

1. 打开与机器人的**私聊**，或在你已添加机器人的**群聊**中
2. 点击输入框旁的 **「+」→ 文件**
3. 选择本地的 **`.pptx`** 文件并发送  
   > 请使用 **.pptx**（Office 2007+）。旧版 `.ppt` 无法自动解析，需另存为 pptx 或改用方式二。
4. 等待 1～3 分钟，机器人会：
   - 自动下载并解析 PPT 正文
   - 按页拆成原子知识点写入知识库
   - 回复入库数量与复习安排（**约 1 小时后**首次复习）

### 可选：指定页码范围

上传后紧接着发一条文字：

```
只学第 3-8 页，帮我入库
```

或：

```
这一份 PPT 我只关心「记忆策略」那几页
```

---

## 方式二：粘贴文字（图片页 / 解析失败时）

若 PPT 以图片为主、解析为空，或上传失败，可粘贴内容：

```
【PPT：章节名】
第1页：……
第2页：……
请按页入库
```

机器人同样会原子化拆分，`source=ppt`。

---

## 方式三：边问边入库

上传 PPT 后，针对某一页提问：

```
第 5 页里的「提取策略」是什么意思？顺便入库
```

讲解中的要点会以 `source=qa` 或补充 `source=ppt` 入库。

---

## 入库后会发生什么

| 时间 | 行为 |
|------|------|
| 入库当下 | 知识点进入活跃库，**1 小时后**首次复习 |
| 复习 pass | 间隔按 1h→2h→4h→…→**最长 48h** 递进 |
| 你说「不会」 | **1 小时**后同一知识点再考 |
| 到期 ≥3 点 | 可发「复习」开始交错出题 |

---

## 常见问题

**Q：支持 PDF 吗？**  
当前仅自动解析 `.pptx`。PDF 请复制文字后按方式二发送。

**Q：文件太大怎么办？**  
正文超过约 12000 字会截断；可分批上传或指定页码分次入库。

**Q：如何确认已入库？**  
机器人回复中会说明入库条数；也可发「我有多少知识点」让 Agent 汇总。

**Q：权限报错？**  
在飞书开放平台 → 权限管理 → 开通 **im:resource**，重新发布应用。
