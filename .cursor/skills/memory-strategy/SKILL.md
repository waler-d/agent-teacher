# 记忆策略（叶修体系）

基于叶修《学习的逻辑》第七章「记忆策略——从记忆的原则到执行方法」。

## 核心框架

记忆策略由三层组成，按优先级：

1. **提取策略** — 适用于一切记忆内容的超级原则（输出 > 输入）
2. **分散记忆** — 提高记忆效率的时间安排法
3. **记忆曲线应用** — 复习频率的方向性指导（勿机械套用数字）

与**脉冲策略**的边界：越零散、越无逻辑、越强调记忆的内容 → 越适合分散记忆；越需理解、逻辑性强的内容 → 越适合脉冲策略（见时间策略 skill 或 reference.md）。

## 何时启用

用户出现以下需求时，按本 skill 回应：

- 背不下来 / 背了就忘 / 复习低效
- 制定单词、课文、公式、知识点背诵计划
- 询问艾宾浩斯表格是否要用
- 想把碎片时间用于复习

## 诊断低效习惯

先识别用户当前做法，对照下表纠偏：

| 低效做法 | 应改为 |
|---------|--------|
| 反复朗读、反复看材料 | 半熟即开始遮盖提取 |
| 背不出立刻翻书 | 先强行回忆至明显费力（约 10–20 秒，因人而异） |
| 复习时直接翻课本朗读 | 先回忆要点，能完整回忆的跳过 |
| 周末集中 2 小时背单词 | 拆成多次短时段分散记忆 |
| 纠结艾宾浩斯精确间隔 | 按「当天→次日→一周内→期中期末」方向安排 |
| 未理解直接硬背 | 先理解再记忆（有意义内容忘得更慢） |

## 工作流：为用户定制记忆方案

### Step 1 — 分析材料

确认：

- **类型**：零散无逻辑（单词、孤立事实）/ 有结构（诗歌、段落）/ 需理解（公式、概念）
- **长度**：短（一次性提取）/ 长（分块提取）
- **场景**：初次背诵 / 复习 / 考前冲刺

### Step 2 — 选择策略组合

```
材料特性判断
├── 零散、无逻辑 → 分散记忆（间隔尽量开）
├── 有结构、中等长度 → 提取策略 + 适度分散
└── 需理解 → 先理解 + 提取策略；复习节奏可更疏

初次背诵 → 提取策略三步法（见下）
复习阶段 → 提取式复习 + 按曲线安排频率
碎片时间 → 纯提取（不需书本）
睡前     → 5–10 分钟提取当天所学（高杠杆细节）
```

### Step 3 — 输出记忆计划

使用下方模板，填入具体材料、时段、次数：

```markdown
## 记忆计划：[材料名称]

**材料类型**：[零散/结构化/需理解]
**策略组合**：[提取策略 + 分散记忆 + 复习节奏]

### 初次背诵（提取策略三步）
1. 读 [N] 遍至半熟（勿读太熟）
2. 遮盖 → 提取；卡住处先强行回忆至费力，再翻材料
3. 分 [M] 块循环，直至整段可提取

### 分散安排
- 总时长 [X] → 拆为 [次数] × [每次分钟]，间隔 [具体时段]

### 复习节奏（中等难度参考）
- 当天 / 次日 / 一周内 / 期中 / 期末各提取复习一次
- 难记内容：前两周加 1–2 次额外提取

### 碎片时间提取点
- [走路/排队/乘车等具体场景]

### 睡前（5–10 分钟）
- 提取当天全部相关知识点，不翻书先回忆

### 精力保障
- 频繁提取需保证睡眠、饮食、锻炼；初期费力感会随练习减弱
```

## 提取策略 — 执行要点

### 意识转换（必强调）

尝试提取**本身就是记忆**，不是「背会了再测试」。初期会累、会沮丧，这是正常现象。

### 初次背诵三步

1. 读几遍形成熟悉感（半熟即可）
2. 遮盖材料，大脑提取；背不出处**不要立刻翻书**，先强行回忆至明显费力
3. 在卡住处翻材料，继续提取直至完成；长材料分块，短材料一次提取

### 复习时使用

1. 不直接朗读/浏览课本
2. 先回忆有哪些内容、哪些要点
3. 能完整回忆 → 已掌握，跳过
4. 卡住 → 同样先忍一会儿再翻书

### 碎片时间示例

走路 5 分钟回忆单词；排队 3 分钟提取古诗；乘车 30 分钟回忆本周生物知识点。

## 分散记忆 — 执行要点

- 同样总时长，**多次短时段 > 一次长时段**
- 机制：前摄抑制 + 后摄抑制 → 单次记忆只有首尾高效、中间低效
- **越零散越无逻辑 → 越应分散**（如英语单词）

## 记忆曲线 — 执行要点

三条特征：

1. 先快后慢的遗忘曲线
2. 每复习一次，再次遗忘变慢
3. 内容越有意义/有逻辑，忘得越慢

**中等难度推荐频率**：当天 → 次日 → 一周内 → 期中 → 期末各一次。

- 不必纠结精确天数表格
- 难记内容：前两周多加 1–2 次（早期复习效率更高）
- 遗忘速度：无意义音节 > 单词 > 诗歌 > 数学公式

## 回应原则

1. **先纠习惯再排计划**：多数问题源于输入式重复，而非「记忆力差」
2. **强调费力感**：提取时的疲惫是有效信号，不是失败
3. **具体可执行**：给出分块方式、时段拆分、复习日历，避免空泛建议
4. **不替代理解**：公式、概念先懂再背
5. **关联精力管理**：高频提取必须配睡眠与锻炼
